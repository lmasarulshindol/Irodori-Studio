## 行為本体の共通フロー

```text
phase 2  入浴 → 口奉仕 → 口内射精（1回目の射精）
phase 3  指愛撫 → 指絶頂（★感じやすい子は連続2回）
phase 4  生挿入 → 挿入中絶頂（★感じやすい子のみ）→ 射精時同時絶頂 → 中出し（2回目の射精）
phase 5  余韻
```

## フォーマット

各台本は `docs/voice-script-sample-filled.md` の構成に準拠しています。

- `## キャラクター: char_xxx（表示名: ...）`
- `### VoiceDesign 用キャプション（長文・本番想定）` — 声質・年齢・演技設計の長文記述
- `### キャプション` — 短（一覧用）／長（説明文用）
- `### セリフ` — `# / phase / voice_tags / 本文` の表形式

## phase の意味

| phase | 段階 | 主感情例 |
|-------|------|---------|
| 1 | 導入 | 戸惑い、期待、静かな高揚 |
| 2 | 近づき・口奉仕 | 照れ、からかい、非言語の吐息 |
| 3 | 高まり・指愛撫→絶頂 | 委ねる、焦り、素直な喘ぎ、指絶頂 |
| 4 | 頂点・挿入→中出し | 痛み→慣れ→感じ→挿入中/射精時同時絶頂 |
| 5 | 余韻 | 安心、恥じらい、仮面再構築 |

## voice_tags の凡例

| タグ | 意味 |
|------|------|
| `whisper` | 囁き・近接感 |
| `breath_heavy` | 息多め・間が短い |
| `pace_slow` | テンポ遅め・余韻 |
| `pitch_high` | 基本の高さ（高め・年少寄り） |
| `pitch_low` | 基本の高さ（低め・大人びた） |
| `broken` | 言葉が途切れる |
| `cry` | 涙声・泣きが混じる（絶頂時） |
| `muffled` | 口を塞がれた状態（口奉仕中） |

## 収録時の方針

- 本文と台本のセリフは**完全一致**（本文で欠けている行があれば台本に準じて本文を修正）
- 各話の「仮面」と「素」の揺らぎ（phase推移）を優先してテイクを取る
- 処女喪失シーン（第1・3・4・5話）では挿入時の痛み・掠れ・涙声を忠実に
- 楓（第2話）は一切の過剰演技を避け、低声・沈黙・呼吸で濃度を作る
- 花音（第4話）は連続絶頂・ハミング・癒し系語尾「〜だよぉ」を特に複数テイク確保
- 澪（第5話）は分析口調から崩壊までの段階推移を phase 別に録り分ける
- 口奉仕（phase 2後半）は非言語中心（「んっ」「ぅっ」「ごくん」等）。TTS では `breath_heavy` + `muffled` の組み合わせで対応
- 指絶頂・射精時同時絶頂は `broken, breath_heavy, cry` を基本として、キャラ基音を重ねる
