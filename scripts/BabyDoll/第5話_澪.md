### VoiceDesign 用キャプション（長文・本番想定）

十三歳の少女の声。基音は淡く高いが、息の通り道が細く、常に一段落ち着いたトーンでコントロールされている。お嬢様育ちの上品な滑舌と、お客を観察する知的な冷たさが同居する。普段は抑揚を殺した敬語で感情の起伏を見せない。今夜は「初めてコース」の夜であり、会話が進むごとにその冷静の仮面に微細なひびが段階的に入っていく。入浴までは抑えたお嬢様口調。口奉仕中は話せず、非言語の鼻息・吐息・嚥下音と、ごく短い気遣いの囁きだけ。口内射精直後、敬語の殻が薄く崩れ「あつい、です」と素の吐露が漏れる。指愛撫の絶頂で「わたし、おかしく、なります」「あ……もう、むり……むりぃ……っ」と敬語がほどけ、初めての絶頂へ倒れ込む。挿入の瞬間「くっ……せ、先……痛、い……っ」「……痛い……」で敬語が完全に剥がれ、十三歳の素の声に戻る。以後は敬語の残滓すら途切れ、「……やめ、ないで……」と裸の懇願だけが残る。挿入中絶頂の直前「いや……いや、先生、なんで、これ……っ」で錯乱し、射精時同時絶頂では「せんせ……わたし、先生の、ものに……」「先生のに、なる……っ」「せんせ……せんせいっ……!」と敬語が完全消失したまま、名前も丁寧さも失って所有と合一を告げる。中出し後の「あったかい、です」で敬語の殻だけが薄く戻り、仮面を結び直す最後の糸になる。余韻では疲労の柔らかさと、仮面を再構築しようとする皮肉の気配が微かに戻る。距離感は常に近めのマイクで、囁きと吐息の比率が高い。

### キャプション

- **短（一覧用）**: アイスブルーの瞳と、三度観察者の椅子から降りた、震える十三歳。
- **長（説明文用）**: 十三歳・お嬢様・知的キャスト「ミオ」。冷静な敬語で自分の情動を翻訳してきた少女が、「初めてコース」で自ら観察者の椅子から降りる夜。序盤は抑揚のない冷静なお嬢様口調。口奉仕中は非言語・嚥下と、ごく短い気遣いの囁きだけ。口内射精直後に一度だけ「あつい、です」と素が漏れる。指絶頂で「わたし、おかしく、なります」「もう、むり……むりぃ……っ」と敬語の殻が大きく崩れる。挿入瞬間で「くっ……せ、先……痛、い……っ」「……痛い……」と敬語が完全に剥がれ、十三歳の素の声に戻る。挿入中絶頂で観察者の役割を完全放棄、射精時同時絶頂で「先生のに、なる……っ」「せんせ……せんせいっ……!」と敬語が消失したまま全身で受容を告げる。中出し後の「あったかい、です」だけが、仮面を結び直す最後の敬語として残り、余韻では皮肉の微かな復帰と、制服に戻る少女の輪郭の再構築で締める。

### 固有ディテールの反映

- **アイスブルーの瞳の冷たさ**: phase 1〜2a は「目で測る声」（抑揚を殺した敬語）で再現する
- **蝶の刺青（背景設定）**: 左手首の蝶の刺青は設定として背景に置くのみで、セリフで直接メタファー化はしない。読者／視聴者が地の文・視覚描写で気付く仕掛け
- **お嬢様の仮面と十三歳の素（敬語崩壊アーク）**: 台本全体で最も感情が露出するのは phase 4a の #44「くっ……せ、先……痛、い……っ」と #46「……痛い……」、そして phase 5 #66「私、13歳」の三連。敬語の「です」だけが phase 4a #43 までは仮面の残滓として張り付くが、#44 以降〜#63 までは敬語すら完全に消失し、#65「あったかい、です」で一度だけ「です」が戻ってきて仮面を結び直す
- **口奉仕中の muffled 区間**: phase 2b の muffled 区間は鼻息・吐息の非言語を主体に、極短い囁きで「きもち、いい、ですか……？」「お口に、どうぞ……」といったシンプルな気遣い／許可の一言だけを差し挟む。学術語彙ではなく身体感覚と気遣いで、観察者が快感に巻き込まれていく過程を表現する
- **敬語の段階的崩壊**: phase 2b→2c で「あつい、です→ごくん→飲み、ました→あたま、くらくら、します→うまく、言葉が、出ません→あたまが、ぼうっと、します→観察、できません→あまえたく、なります→感じたい、です」と、お嬢様敬語が身体感覚の素の吐露にゆっくり溶けていく。phase 3 終盤の #39「もう、むり……むりぃ……っ」でついに敬語の語尾「〜ます」が「むりぃ」という幼い響きに退化する
- **観察放棄のアーク**: phase 3 #25「観察、できません……先生」（口奉仕直後・指愛撫序盤の最初の放棄宣言）／ phase 4a #42「たぶん、途中までは」（挿入前の観察継続の自己否定）／ phase 4b #54「いや……いや、先生、なんで、これ……っ」（挿入中絶頂直前の錯乱による放棄）／ phase 4c #62「せんせ……せんせいっ……!」（射精時同時絶頂での名前と観察の完全消失）。観察の語彙は phase 3 以降再び出てこず、身体感覚と名前呼びだけが残る
- **名前崩壊（せんせ／先生）**: phase 4c #60「せんせ……わたし、先生の、ものに……」と #62「せんせ……せんせいっ……!」でのみ「先生」→「せんせ」への欠落を使う。他の区間では「先生」のまま保持し、崩壊の極みだけで使い分ける
- **献身の形**: phase 2c #27「もっと、先生のこと、感じたい、です」と phase 5 #67「先生のこと、忘れられ、ない、です」が呼応。観察者ではなく、ひとりの少女として先生だけを覚えてしまった、という静かな着地

### 関係性フレーズの固定

- 二人称: 常に **「先生」**（指名客の久世征一郎への呼称）。phase 4c #60・#62 のみ崩壊形「せんせ」を使用
- 一人称: 常に **「私」**（お嬢様らしい一人称を崩さない）。phase 3 #38、phase 4c #60 のみ「わたし」と平仮名化し、敬語の殻が割れた瞬間を表現
- 自称の崩壊は一切ない。仮面が崩れても「私／わたし」「先生／せんせ」のまま。崩れるのは語尾と息のほうである
- 敬語の「です」「ます」は phase 4a #44 で完全に途切れ、#65「あったかい、です」で一度だけ戻る。完全な崩壊時は「……っ」で語尾が欠落する
