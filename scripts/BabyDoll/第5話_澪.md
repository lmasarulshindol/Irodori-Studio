### VoiceDesign 用キャプション（長文・本番想定）

十三歳の少女の声。基音は淡く高いが、息の通り道が細く、常に一段落ち着いたトーンでコントロールされている。お嬢様育ちの上品な滑舌と、お客を観察する知的な冷たさが同居する。普段は抑揚を殺した敬語で感情の起伏を見せない。今夜は「初めてコース」の夜であり、会話が進むごとにその冷静の仮面に微細なひびが段階的に入っていく。入浴までは抑えたお嬢様口調。口奉仕中は話せず、非言語の鼻息・吐息・嚥下音と、ごく短い気遣いの囁きだけ。口内射精直後、敬語の殻が薄く崩れ「あつい、です」と素の吐露が漏れる。指愛撫の絶頂で「わたし、おかしく、なります」と敬語の残骸だけを残して初めての絶頂へ倒れ込む。挿入の瞬間「先生、痛い、です」で十三歳の素の声に戻り、以後「先生」「私」「です」だけが仮面の最後の残滓として語尾に張り付く。挿入中絶頂で観察者の役割を完全放棄し、射精時同時絶頂で「先生のものに、なります」と全身で受容を告げる。中出し後の「あったかい、です」は、敬語の殻だけが薄く残った、ほとんど素の声。余韻では疲労の柔らかさと、仮面を再構築しようとする皮肉の気配が微かに戻る。距離感は常に近めのマイクで、囁きと吐息の比率が高い。

### キャプション

- **短（一覧用）**: アイスブルーの瞳と、三度観察者の椅子から降りた、震える十三歳。
- **長（説明文用）**: 十三歳・お嬢様・知的キャスト「ミオ」。冷静な敬語で自分の情動を翻訳してきた少女が、「初めてコース」で自ら観察者の椅子から降りる夜。序盤は抑揚のない冷静なお嬢様口調。口奉仕中は非言語・嚥下と、ごく短い気遣いの囁きだけ。口内射精直後に一度だけ「あつい、です」と素が漏れる。指絶頂で「わたし、おかしく、なります」と敬語の殻が大きく崩れる。挿入瞬間で「先生、痛い、です」と十三歳の素の声に戻る。挿入中絶頂で観察者の役割を完全放棄、射精時同時絶頂で「先生のものに、なります」と全身で受容を告げる。敬語の「です」と「先生」「私」だけが最後まで残る。余韻では皮肉の微かな復帰と、制服に戻る少女の輪郭の再構築で締める。

### 固有ディテールの反映

- **アイスブルーの瞳の冷たさ**: phase 1〜2a は「目で測る声」（抑揚を殺した敬語）で再現する
- **蝶の刺青（背景設定）**: 項背の蝶の刺青は設定として背景に置くのみで、セリフで直接メタファー化はしない。読者／視聴者が地の文・視覚描写で気付く仕掛け
- **お嬢様の仮面と十三歳の素**: phase 4a「先生、痛い、です」と phase 4c「私、13歳」二連が台本全体で最も感情が露出する瞬間。敬語の「です」だけが仮面の残滓として残り、他は完全に素
- **口奉仕中の muffled 区間**: phase 2c の muffled 区間は鼻息・吐息の非言語を主体に、極短い囁きで「先生の、大きい……」「きもち、いい、ですか……？」といったシンプルな感想・気遣いを差し挟む。学術語彙ではなく身体感覚と気遣いで、観察者が快感に巻き込まれていく過程を表現する
- **敬語の段階的崩壊**: phase 2c→2d で「あつい、です→ごくん→飲みました→のど、あつい→あたま、くらくら→うまく言葉が出ません→あたまが、ぼうっと→あまえたく、なります」と、お嬢様敬語が身体感覚の素の吐露にゆっくり溶けていく
- **観察放棄の三度**: phase 2d #30「観察、できません……先生」（口奉仕後の最初の放棄宣言）／ phase 4b #67「先生、観察、できません」（挿入中絶頂の直前）／ phase 4c #78「もう、観察、しません」（射精後の最終放棄）。三段で「観察者の椅子から降りる」過程を完成させる
- **献身の形**: phase 2d #32「もっと、先生のこと、感じたい、です」と phase 4c #84「先生のこと、忘れられ、ない、です」が呼応。観察者ではなく、ひとりの少女として先生だけを覚えてしまった、という静かな着地

### 関係性フレーズの固定

- 二人称: 常に **「先生」**（指名客の久世征一郎への呼称）
- 一人称: 常に **「私」**（お嬢様らしい一人称を崩さない）。phase 4b #59 のみ「わたし」と平仮名化し、敬語の殻が割れた瞬間を表現
- 自称の崩壊は一切ない。仮面が崩れても「私」「先生」のまま。崩れるのは語尾と息のほうである
- 敬語の「です」「ます」は最後まで残る。完全な崩壊時のみ「……っ」で語尾が欠落する
