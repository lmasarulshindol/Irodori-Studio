# BabyDoll 初接客オムニバス 第2話 紅の支配者

> 対応本文: `999_その他/N000001/001_BabyDoll/本文/第2話_紅の支配者.md`
> 視点: 梶原 恭一郎（52歳・プラチナ会員）／ コース: VIPスペシャル（120分）
> 【改訂版】行為本体フローは「入浴 → 口奉仕 → 口内射精 → 指愛撫 → 指絶頂（声を出さない） → 挿入（経験者） → 射精時同時絶頂（最小表現） → 中出し → 余韻」に準拠。
> 【拡張版】phase 2（口奉仕）と phase 4（本番＝挿入・中出し）に、最小単位の対話セリフと息のみの非言語行を追加挿入。喘ぎは増やさず「沈黙の支配」を維持。

## キャラクター: char_kaede（表示名: 桐原 楓 ／ 源氏名: カエデ）

### VoiceDesign 用キャプション（長文・本番想定）

十四歳、BabyDollの最年長キャストにしてVIPキャストデビュー初日。漆黒のストレートロングに深紅のインナーカラーを一筋。切れ長のダークブラウンの目。深紅のシルク調ベビードールをまとい、背中は肩甲骨まで大きく開いている。身長一五四センチで年齢のわりに大人びた体つき、左鎖骨に小さな黒子。接客スタイルは「無言の支配」。声は基本的に低く、静かで、遅い。一本一本の台詞の間が長く、発声の前に呼吸の気配すら感じさせない抑制がある。声量は極端に小さく、マイクに近接した囁きと、呼吸の音の間を行き来する。本話の改訂版では、口奉仕中は非言語の「んっ」「ぅ……」のみ、指で絶頂させられる瞬間は声を出さず肩の震えと一筋の「……っ」のみ、挿入から射精時同時絶頂までも「ん……」の最小単位で抑え切る。全編を通じて喘ぎを放出しない姿勢を維持し、代わりに呼吸の間合いと沈黙で支配する設計。感情を表に出さないが、鎖骨の黒子に触れられた瞬間、聞かれてはいけない過去を察された瞬間にだけ、十四歳の少女としての揺らぎが、わずかに声を掠れさせる。基本トーンは whisper + pitch_low + pace_slow。

### キャプション

- **短（一覧用）**: 沈黙が支配する、十四歳の紅い静寂。二度の絶頂も、声を出さない。
- **長（説明文用）**: 深紅のシルクと湯気のVIPルームで綴る、玄人キャストの正規デビュー初夜。台詞は極端に少なく、発声は低く、テンポは遅い。ほとんどの時間を呼吸と視線で埋め、ごく短い数行の台詞に「支配」「受容」「聖域」のすべてを凝縮させる。口奉仕の非言語、指絶頂の一筋の震え声、射精時同時絶頂の最小「ん……」、そして鎖骨の黒子に触れられた一行の揺らぎ——その四つの微細な波形で、十四歳の完成されたプロの一夜を描く。

### 演出メモ

- **phase 1（#1〜#3）**: 発声は最小限。一行言い切るごとに1〜2秒の無音を置く。威圧ではなく「場の温度を下げる」沈黙。
- **phase 2 入浴・口奉仕導入（#4〜#6）**: #4 で客の発話権を封じる。#5「……いいよ。」が口奉仕に入る前の唯一の許可——膝をつき、相手の前に身を低くしながらの一言。承認以外の感情は乗せない。#6「……ふ。」は口奉仕開始直前の一息。
- **phase 2 口奉仕本体（#7〜#13）**: 口を塞がれた状態の非言語行。muffled を強めに。喘ぎ声ではなく「抑制された、気圧の変化程度の漏れ」——「ぅ……っ／ん……／んっ」の差異は、呼吸量のわずかな違いとしてのみ現れる。#10「……ん。」は奉仕中の微かな合図——客の手や身体の動きへの最小の応答。
- **phase 2 口内射精〜余韻（#14〜#18）**: #14 は嚥下音のみ。表情は変えない。#15「……ごく……。」は口内に残ったものを二度目に飲み込んだ最小の報告——口の端の白を指で拭う仕草を、声に乗せる必要はない。#16「……ん。」は飲み込み切った直後の受容音。#17〜#18 の「……ふ。／……ふ……。」は呼吸が静まっていく二段の息——口を離してから視線を上げるまでの沈黙を、息だけで埋める。
- **phase 3 指愛撫〜指絶頂（#19〜#23）**: #19「……ふ。」は指愛撫開始の最初の一息、#20「っ……。」は波の兆し、#21 が本台本で最大の「崩壊」——ただし、それでも他キャラの半分以下の音量・明確度。「……っ」のみで、肩の震えだけで絶頂を伝える。#22 は絶頂直後の、一度だけ漏れる呼吸。#23 は呼吸を静めた直後の受容音。
- **phase 4 挿入前（#24〜#26）**: #24「……二度目は、ゆっくり。」が本話二つ目の発話。#25「……どうぞ。」は挿入の許可——客の動きを止めず、自分から差し出す静かな一言。#26「……ふ。」は挿入を待つ息のバッファ。
- **phase 4 挿入瞬間〜慣れ（#27〜#31）**: #27「……っ。」は挿入瞬間の最小反応、#28〜#29 は慣れとしての呼吸。#30〜#31 の「……ん。／……ん。」は受容を表す二段の最小合図。
- **phase 4 挿入中盤の合図（#32〜#33）**: #32「……中で。」は中出しを許可する最小の二音節——「中で済ませて構わない」という承認だけを、単語ではなく句読点だけで伝える本話の核。#33「……ん……。」のバッファで再び沈黙へ戻る。
- **phase 4 射精時同時絶頂（#34〜#37）**: #34「……っ、ん……」、#35「……っ。」、#36「……ん。」、#37「……ん……っ。」の四段最小表現。声量は #21 と同等まで抑える。射精承認の「……っ」と「……ん」は息のみで、絶頂の自覚を客に「気づかれない」のが楓の支配。
- **phase 4 中出し直後（#38〜#41）**: #38「……ふ。」、#39「……はぁ……。」、#40「……ん……。」、#41「……はぁ。」の四音で呼吸がゆっくり戻っていく——中出しを受け切った身体が静かに鎮まっていく息の流れ。
- **phase 5（#42〜#45）**: #42 だけがこの台本の「素」の一行。声を一段さらに落とし、語尾を微かに揺らす（`broken`）。#43 でまた仮面を被り直すが、声のコアが #1〜#2 より0.5段ほど柔らかい。#44「……また、どうぞ。」は素と仮面の中間——別れ際の最小の挨拶、相手の存在をかすかに肯定する一言。#45 は完全な接客モードに復帰した別れの定型。
- **呼び方の統一**: 客への呼称は「梶原さま」。自称は使わない（台本内で一度も主語を出さない設計）。
- **全体量**: 45行。発声行の大半は「……」と息のみで構成され、行数そのものが楓の「沈黙の支配」を表現する。新規追加分も最大限「息のみ」「最小音節」に抑え、対話セリフは全話で4個（#5「……いいよ。」／#25「……どうぞ。」／#32「……中で。」／#44「……また、どうぞ。」）に制限。

## 備考

- 本文中のセリフと台本のセリフは一字一句一致。TTS尺確保および対話・献身表現拡充のため phase 2〜4 に最小単位の非言語行と短い対話を追加挿入した拡張版。新規行はすべて「……」「ふ」「っ」「ん」「ごく」「いいよ」「どうぞ」「中で」「また、どうぞ」等の最小音節で構成し、楓の「無言の支配」の姿勢を崩さない。
- 楓は本話を通じて「プロの初日」として描かれるため、処女喪失系の痛み声・涙声は含まない。
- #7〜#13 は「一度目の射精 = 口内射精」に至る口奉仕の非言語行。ユニットとしてまとめて演じる。
- #34〜#37 は「二度目の射精 = 中出し」と同時の、楓自身の絶頂の最小表現。二度の絶頂（#21 と #34-37）はいずれも無声に近いが、#34-37 の方がわずかに息の長さがある。
- 追加された対話セリフ（#5・#25・#32・#44）はいずれも「許可・承認」の最小単位として機能し、楓の「献身」を「能動的な放出促進」ではなく「沈黙による静かな許諾」として表現する。莉子の「ぜんぶ、ちょうだい」型の積極的献身とは正反対の設計。
- 呼吸制御とト書き上の間合いが本作の主役。台本は骨格のみで、演出側の余白設計を前提とする。
